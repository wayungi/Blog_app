## Getting Started
- Install  Ruby
- Install postgresql.
- clone the repo with  `git clone git@github.com:wayungi/rails_blog.git`

- `cd rails_blog`
- Run `bundle install`
- Run `rails s`
- Open Browser `http://localhost:3000/`

- Run `rails db:create`
